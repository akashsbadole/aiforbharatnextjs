## 📋 Overview

**Swasthya Mitra** (Health Friend) is a comprehensive, AI-powered health platform designed specifically for India's diverse healthcare needs. It combines cutting-edge AI technology with grassroots healthcare delivery, supporting 8 Indian languages and serving citizens, health workers, doctors, and administrators.

### 🎯 Mission

To democratize healthcare access across India by bridging the gap between rural communities and quality healthcare through AI-powered assistance, telemedicine, and community health networks.

## 🌟 Features

### 1. 🤖 AI Health Assistant (7-Level Intelligence)

The world's first 7-level intelligent health AI assistant:

| Level | Name | Description |
|-------|------|-------------|
| L1 | **Basic AI** | Direct question answering with empathy |
| L2 | **Context-Aware** | Remembers conversation history & patient context |
| L3 | **Tool-Using** | Access to databases, APIs, and real-time data |
| L4 | **Reasoning** | Multi-step problem analysis and planning |
| L5 | **Multi-Agent** | Specialized AI agents collaborate (Triage, Diagnosis, Emergency, etc.) |
| L6 | **Domain-Expert** | Medical knowledge base with RAG (Retrieval-Augmented Generation) |
| L7 | **Self-Improving** | Learns from feedback and adapts responses |

**Capabilities:**
- 🎤 Voice input support (Speech-to-Text)
- 🖼️ Image analysis for skin conditions & reports
- 🗣️ Text-to-Speech for responses
- 🌐 Multi-language support (8 Indian languages)
- 📊 Structured medical responses with:
  - Immediate Steps
  - Possible Causes
  - Red Flags
  - Home Remedies
  - Prevention Tips
  - Nearby Facilities
  - Emergency Numbers

## 🏗️ Architecture

### Technology Stack

| Category | Technology |
|----------|------------|
| **Frontend** | Next.js 16, React 19, TypeScript |
| **Styling** | Tailwind CSS, shadcn/ui |
| **Backend** | Next.js API Routes |
| **Database** | SQLite with Prisma ORM |
| **AI/ML** | Google Gemini API (LLM, Multimodal, TTS, ASR) |
| **State** | Zustand, TanStack Query |
| **Auth** | NextAuth.js v4 |

## 🚀 Installation

### Prerequisites

- Node.js 18+
- Bun runtime
- SQLite

### Setup

```bash
# Clone the repository
git clone https://github.com/your-org/swasthya-mitra.git

# Navigate to project
cd swasthya-mitra

# Install dependencies
bun install

# Setup database
bun run db:push

# Start development server
bun run dev
```

### Environment Variables

```env
DATABASE_URL="file:./dev.db"
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"
```
